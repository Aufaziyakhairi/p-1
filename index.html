Tentang

Saya adalah siswa SMK Negeri 1 Ciomas jurusan Teknik Komputer dan Informatika yang memiliki minat dan keahlian dalam bidang pengembangan game. Saat ini saya sedang mempersiapkan diri untuk melaksanakan Praktik Kerja Lapangan (PKL) sebagai bagian dari program pendidikan dan pengenalan dunia industri. Saya dikenal sebagai pribadi yang disiplin, bertanggung jawab, dan memiliki semangat belajar yang tinggi. Dengan bekal yang saya miliki, saya siap untuk berkontribusi serta terus mengembangkan kemampuan saya dalam lingkungan kerja profesional.

HTML

Saya memiliki pengetahuan dasar dalam bahasa pemrograman HTML, termasuk kemampuan untuk membuat struktur dasar halaman web.

Godot Engine

Saya mulai mempelajari Godot Engine dan telah mencoba membuat beberapa proyek sederhana. Saya memahami konsep dasar seperti node system, dan scene tree

Microsoft Word

Saya memiliki kemampuan dasar dalam menggunakan Microsoft Word, seperti membuat dan mengedit dokumen, mengatur format teks dan paragraf, menyisipkan gambar dan tabel, serta mengatur tata letak dokumen secara rapi. Saya juga memahami penggunaan fitur-fitur umum seperti header, footer, dan penomoran halaman untuk kebutuhan penyusunan laporan atau tugas sekolah.

Microsoft Excel

Saya sedang belajar menggunakan Microsoft Excel dan sudah bisa membuat tabel, menghitung data dengan rumus dasar, serta memahami cara menyusun data secara terstruktur. Walaupun masih tahap awal, saya cukup terbiasa menggunakan Excel untuk mengolah informasi secara sederhana.
